## Testing Workflow

1. **Customer Flow:**
   - Login as a customer (e.g., john@example.com)
   - Go to "Book Session"
   - Select teacher, date, and time
   - Submit booking request

2. **Teacher Flow:**
   - Login as a teacher (e.g., sarah@corestudio.com)
   - View pending booking requests
   - Confirm or reject bookings
   - Mark attendance for completed sessions

3. **Admin Flow:**
   - Login as admin (admin@corestudio.com)
   - View all pending bookings across all teachers
   - Confirm or reject any booking
   - View system statistics
